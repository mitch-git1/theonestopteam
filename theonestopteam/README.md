# The One Stop Team – Kodi Repository

Welcome to the **The One Stop Team Kodi Repository** 👋  
This is the official third-party Kodi repository for addons developed and maintained by **The One Stop Team**.

This repository focuses on **lightweight, Kodi-native service addons** designed to enhance playback behavior and user experience without interfering with normal Kodi operation.

---

## 📦 Available Addons

### ▶️ The EDL Skipper
**Service Addon**

Skip intros and outros using `.edl` timestamp files.

**Key features:**
- Reads EDL files placed alongside media
- Skip Intro / Skip Outro popups
- Optional automatic skipping
- Designed for TV show playback and playlists
- Uses Kodi-native dialogs and controls

Ideal for users who already maintain EDL files or want precise control over skipping intros and outros.

---

### ▶️ The Playback Watcher
**Service Addon**

Helps prevent endless binge-watching by checking if you are still watching.

**Key features:**
- Prompts after a configurable number of episodes
- Options to:
  - Continue watching
  - Stop playback
  - Snooze the prompt
- Playback is paused while the prompt is shown
- Optional automatic stop if the prompt is ignored

Designed to be unobtrusive and fully Kodi-native.
